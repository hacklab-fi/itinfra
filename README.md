# IT-infra

Document hacklab.fi common use infra

## Matrix server

Documentation for hacklab.fi usage, Matrix-server and other things

This repository is source, actual site is here:
[Hacklab.fi technical documentation](https://docs.hacklab.fi)

The site is build with mdbook and you can run your own local version by:

* installing mdbook and git
* cloning this repo with `git clone git@github.com:hacklab-fi/itinfra.git`
* running `mdbook serve`

Feel free to make pull requests to make the data even better for everybody to use
