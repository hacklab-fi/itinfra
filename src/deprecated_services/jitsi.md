# Jitsi - Deprecated

Hacklab had jitsi running for video calls for few years. It was used extensively to allow remote access to meetings and for other streaming purposes. It was deprecated in 2025 and replaced by Element call

# Old content for history

Jitsi (from Bulgarian: жици — "wires") is a collection of free and open-source multiplatform voice (VoIP), video conferencing and instant messaging applications for the Web platform, Windows, Linux, macOS, iOS and Android.
